ℹ️ About This Project

This AI-driven deepfake detection system uses advanced deep learning models to identify manipulated media:

Image Detection

Uses ResNet-18 architecture to analyze individual frames and classify them as real or fake with high accuracy.

Video Detection

Combines ResNet-18 feature extraction with LSTM (Long Short-Term Memory) networks to analyze temporal patterns across 16-frame sequences, providing more robust deepfake detection for video content.

Technology Stack

Built with PyTorch, Flask, OpenCV, and cutting-edge deep learning techniques. Supports real-time detection from webcam and file uploads.
